Prequisites:
    A working python 2 installation (python 3 probably also works) with numpy.

Running:
	To confirm the program works correctly, please use
		python ROLF.py
	this will create a ROLF Net based on the data in test.data. 
	The final center location will be printed to test.out, using one
	line for each center in the following format:

	x1 x2 x3 ... xn        size

Documentation:
	Further documentation can be found in ROLF.py itself.
